## Installation

1. Add the following to your composer.json and run `composer update`

    ```json
    {
        "require": {
            "schickling/queue-checker": "dev-master"
        }
    }
    ```

2. Add `Schickling\QueueChecker\QueueCheckerServiceProvider` to your config/app.php
